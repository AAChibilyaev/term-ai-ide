# Кастомизация интерфейса

## Доступные темы
1. **Dark Modern** - темная тема с акцентами
   ```bash
   theme apply dark-modern
   ```

2. **Light Minimal** - светлая минималистичная
   ```bash
   theme apply light-minimal
   ```

## Создание своей темы
1. Создайте файл `~/.config/package/theme-custom.yaml`
2. Задайте параметры:
```yaml
colors:
  primary: "#3498db"
  background: "#2c3e50"
```
